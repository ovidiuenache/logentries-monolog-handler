## Installation

Add the following to your composer.json and run `composer update`

```json
{
    "require": {
        "ovidiuenache/logentries-monolog-handler": "v2.3"
    },
    "repositories": [
        {
          "type": "vcs",
          "url": "https://github.com/ovidiuenache/logentries-monolog-handler"
        }
    ]
}
```
